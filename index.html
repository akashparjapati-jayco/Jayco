ℹ Setup Guide
Production Sheet:
• Paste apps_script.js → Run setup → Deploy

Vendor Sheet:
• Paste vendor_apps_script.js → Run setup → Deploy

Both deployments:
• Execute as: Me
• Who has access: Anyone

Default logins:
• admin / admin123 → all modules
• accounts / acc123 → vendor only
• production / prod123 → prod only
• sales / sales123 → sales only

Modules column (Users sheet):
Comma-separated: prod,vendor,sales,admin
Any combination works per user.
